# Zettelkasten

·tools ·notetaking ·knowledge

*Note-taking method* basado en la idea de que cada nota representa un nodo que forma
parte de una red de conocimiento, donde a medida que se va añadiendo información esta
debe ir relacionandose a través de referencias. Este método se basa en una toma de notas
"atómica", es decir, cada nota debe ser capaz de explicarse sin tener que tener las otras
notas, pero al momento de verla dentro del contexto de las otras notas poder generar una
introspección de la información más profunda y útil para la red de conocimiento.

Lo aspectos claves de este método son:

- Notas cortas, autoexplicativas y directas al grano
- Referencia entre notas para generar esta red de conocimiento
- Utilizar *tags* para poder acceder rapidamente a la información
