# Pull all branches from Git repo

~~~
git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done
git fetch --all
git pull --all
~~~

This series of commands does the following:

1. Track all remote branches for given Git repo
2. Fetch all data from given repo
3. Pull all data from given repo
